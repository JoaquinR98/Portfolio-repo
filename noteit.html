Note-It

A minimal multifunctional application for note-taking and personal schedule organization.

Problem

The majority of note-taking applications seem to be oversimplified and do not offer other related functional features that users might find great value in.

Solution

Create a simple, accesible and personalized application for taking notes and keeping track of tasks at any place and time.
